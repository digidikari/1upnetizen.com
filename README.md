# 1UP NETIZEN - MLM Matrix Website

This is a MLM Matrix website hosted on GitHub Pages and powered by Firebase.

## Features

- User Authentication
- Real-time Database
- Responsive Design

## Setup

1. Clone the repository
2. Configure Firebase in `firebase-config.js`
3. Deploy to GitHub Pages
